# ElMino – Halal Supply Chain Smart Contract

A Solidity-based smart contract for halal supply chain tracking, developed for the **Blockchain & Applications** course.

## 📌 Features
- Role-based access control:
  - Admin (Deployer)
  - Producer
  - Halal Authority
  - Distributor
  - Retailer
- Batch creation by producers
- Halal certification by authorized halal authority
- Status updates across supply chain stages
- Ownership transfer between actors
- On-chain traceability of halal products

## 🛠️ Technology
- Solidity ^0.8.20
- Remix IDE
- Ethereum (Remix VM)
